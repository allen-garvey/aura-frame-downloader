# Aura Photo Downloader

Download all the photos in a given [Aura](https://app.auraframes.com) frame.

## Dependencies

* npm
* node >=22

## Getting Started

* `cp .env.example .env`
* Fill out the values in `.env`
* `npm install`
* `npm start`
